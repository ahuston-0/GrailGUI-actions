Detailed Description

Represent a config file used to configure complicated applications

File format supports comments starting with # name-value pairs of data, with declaration of the type of names supported in the constructor so that subclsses of Config can specify the required types of each tag. In this way, the Config utility can display the line number and an error if the data is not as required

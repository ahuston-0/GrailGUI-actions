5 /*
6  relative pointer, constant-size string, light weight
7  Allocate a single static 4Gb chunk of memory in address space
8  and allocate out of that
9  Individual memory is not collected, the policy is free to mark
10  This is ideal for situations where a lot of strings are allocated, and then
11  deallocated all at the same time.
12 
13  This code uses a global pool, so allocation and deallocation is
14  NOT thread safe. As a result, it is BRUTALLY FAST.
15 */
